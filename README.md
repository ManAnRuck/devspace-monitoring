# devspace-monitoring
DevSpace 6 monitoring configuration with prometheus &amp; grafana

## Prerequirements

- [Devspace](https://www.devspace.sh/docs/getting-started/installation)
## Usage

run
```
devspace dev
```

- open [localhost:3000](http://localhost:3000/)
- Login:
    - username: admin
    - password: prom-operator
